RatLang

Requirements
------------

 - modern C compiler
 - flex
 - bison
 - readline

Compiling
---------

### Generate autotools stuff

    ./autogen.sh

### Build binaries

    ./configure
    make
    make install

### Build code package for distribution (optional)
    
    make distcheck
